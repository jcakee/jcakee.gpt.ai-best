# Spin Wheel

A simple web-based spinning wheel project built with HTML, CSS, and JavaScript.

## Features

* Interactive spinning wheel animation
* Random selection result
* Responsive design
* Lightweight and easy to customize

## Purpose

This project was created as a simple demonstration of front-end development using vanilla JavaScript. Users can click the spin button to rotate the wheel and receive a random result.

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Click the spin button and enjoy.

## Future Improvements

* Custom wheel items
* Sound effects
